# elasticsearch-engineer-training-environment

This is a simple local training environment using Vagrant for training of Elasticsearch Engineer I + II Lab lessons.

- [Elasticsearch Engineer I](https://www.elastic.co/de/training/elasticsearch-engineer-1)
- [Elasticsearch Engineer II](https://www.elastic.co/de/training/elasticsearch-engineer-2)

## Requirements

Install the following requirements using apt/brew/choco or with binaries:

- Vagrant
- VirtualBox

## Usage

Start:
```vagrant up```

Stop:
```vagrant down```

SSH:
```
vagrant ssh server1
vagrant ssh server2
vagrant ssh server3
```
